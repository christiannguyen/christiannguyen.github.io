First full-stack project I've decided to do (that I didn't completely give up on). Always wanted to do a full-stack side project but they usually died before it really became something. This one has been pretty fun so far and i like the idea so hope it sees the light of day. Also mainly using this to learn new things (like styled components, diff ways to structuring a frontend & backend app, use fun node modules, and to just get better at coding)

Workflow & Thought-process

Document and write out all the basic features and functionality. That would be the foundation to figuring out the frontend and backend. Then started to design some basic mockups with Sketch and asked some friends for feedback. Backend schema was thought out beforehand for the most part for basic features and functionality. Didn't want to get too advance and thought simple tables would suffice. Some pieces were pretty unorganized (bad on my part), kind of started work all over place at the beginning but started to slow down, get more organized, and focused on finishing each key pieces one at a time. (user accounts, creating a trip, searching for activities, etc)
